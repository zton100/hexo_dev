# 编辑器VScode代码如何自动换行

>打开编辑器, 文件 -> 首选项 -> 设置 -> 文本编辑器 -> 找到word wrap，（滚动鼠标翻起来比较麻烦，直接搜索wrap关键字即可），找到之后设置为ON就可以了。路径大致就是上面这样的，版本不同，可能稍有差别，整体上算是大同小异吧！英文版的编辑器：File -> Preferences -> Settings -> Text Editor -> word wrap
